# Assets Folder

Place your media files here:

## Required Files:

1. **hero-video.mp4** - Your main video editing showreel for the hero section
   - Recommended size: 1920x1080
   - Duration: 10-30 seconds
   - Format: MP4 (H.264)
   - Keep file size under 10MB for faster loading

2. **your-photo.jpg** - Your professional photo for the About section
   - Recommended size: 500x600px
   - Format: JPG or PNG
   - Professional headshot or creative portrait

3. **Project Thumbnails** - Screenshots or stills from your projects
   - Recommended size: 1200x800px
   - Format: JPG or PNG
   - Name them descriptively (e.g., commercial-ad-1.jpg, music-video-1.jpg)

## Optional Files:

- **favicon.ico** - Website icon for browser tab
- **og-image.jpg** - Social media preview image (1200x630px)

## Tips:

- Optimize all images before uploading (use TinyPNG or similar tools)
- Keep total assets folder size reasonable for web performance
- Use consistent naming conventions
- Consider creating different sizes for responsive images
